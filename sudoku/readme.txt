Source code for part 1 and part 2 of the assignment have been written on a system with python version 2.7.17
pysat, csv, random are the python libraries that were imported in the beginning of the source code
Id any of the above mentioned libraries does not exist in the test machine then the libraries can be installed using following commands respectively
--$pip install python-sat
--$pip install csv
--$pip install random
